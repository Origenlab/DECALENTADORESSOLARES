# Guia de Estilo Visual

## Layout

- Contenedor maximo: `1400px`
- Padding horizontal:
  - Mobile: `1rem`
  - Tablet: `2rem`
  - Desktop: `3rem`

## Escala de espaciado

- `--space-xs`: 8px
- `--space-sm`: 16px
- `--space-md`: 24px
- `--space-lg`: 40px
- `--space-xl`: 64px
- `--space-2xl`: 96px

## Tipografia

- Primaria: `Inter`
- Secundaria: `Poppins`
- Jerarquia: `--text-xs` a `--text-5xl`

## Colores

- Primario: `#c41e3a`
- Primario oscuro: `#9a1829`
- Secundario: `#1a1a2e`
- Fondo: `#ffffff`
- Superficie: `#f8f9fa`
- Borde: `#e5e7eb`

## Principios UI

- Cards con borde + sombra ligera.
- CTA claros y consistentes en todos los templates.
- Navegacion sticky para acceso rapido en scroll.
- Contraste AA orientado a legibilidad.
