# Analisis de Referencia: gamademexico.com

## Patrones visuales detectados

- Header en dos niveles: topbar informativa + barra principal sticky.
- Contenedor maximo amplio (`1400px`) con buen aire lateral.
- Hero con alto contraste, CTA principal y mensajes de confianza.
- Bloques por seccion con ritmo de espaciado consistente.
- Tarjetas con borde suave, sombra ligera y hover discreto.
- Footer multi-columna orientado a SEO (enlaces internos + contacto).

## Paleta extraida

- Primario: `#C41E3A`
- Primario oscuro: `#9A1829`
- Secundario: `#1A1A2E`
- Acento neutro: `#F5F5F5`
- Texto principal: `#333333`
- Texto secundario: `#666666`

## Tipografia detectada

- Familia principal: `Inter`
- Jerarquia de titulos con peso alto (700-800)
- Cuerpo en 16px aprox con interlineado amplio

## Componentes reutilizables identificados

- Header + navegacion responsive.
- Tarjetas de servicios y catalogo.
- Secciones de beneficios y pruebas sociales.
- CTA de conversion con boton principal y secundario.
- Footer con estructura de enlaces internos.

## Flujo de usuario observado

1. Entrada por Hero (propuesta de valor + CTA).
2. Validacion de confianza (beneficios/testimonios).
3. Exploracion de servicios o productos.
4. Conversion por WhatsApp o contacto directo.
